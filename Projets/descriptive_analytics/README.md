# Business Analytics Portfolio

This repository contains a collection of **data analytics projects** developed as part of my studies in **Business Analytics and Supply Chain Management at Université Laval**.

The goal of this portfolio is to demonstrate my ability to:

* clean and transform raw datasets
* perform exploratory and descriptive data analysis
* build meaningful visualizations for decision-makers
* structure reproducible analytical workflows

Most projects were developed using **R**, the **tidyverse ecosystem**, and **ggplot2**.

---

# Projects

## 1. Quebec Trees Analysis

This project analyzes **urban tree data from the City of Québec**.

The objective is to identify locations containing the highest number of trees and compute descriptive statistics about different tree types.

Key tasks:

* Data import using `read_csv`
* Handling missing values
* Data transformation using `dplyr`
* Statistical summaries
* Identification of locations with the most trees

Technologies used:

* R
* tidyverse
* dplyr

---

## 2. Chez Tony – Customer Traffic Analysis

This project analyzes **hourly customer traffic data** for a high-tech locksmith store.

The goal is to understand **customer patterns across hours, days, and weeks**, and provide insights to help the owner decide whether to adjust opening hours or hire additional staff.

Key tasks:

* Reading Excel data with `readxl`
* Transforming raw data into tidy format
* Extracting day and week features using `lubridate`
* Visualization of customer patterns using `ggplot2`

Technologies used:

* R
* tidyverse
* readxl
* lubridate
* ggplot2

---

## 3. Weather and Employee Activity Analysis

This project explores the relationship between **weather conditions and remote employee activity**.

The analysis combines **hourly weather data from Environment Canada** with **employee server connection logs**.

Key objectives:

* Load and combine multiple weather data files
* Clean and preprocess environmental data
* Merge weather information with employee activity logs
* Identify potential relationships between weather conditions and work activity

Technologies used:

* R
* tidyverse
* dplyr
* ggplot2
* lubridate
